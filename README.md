## 📖 實驗簡介

本網頁提供兩個完整的密度實驗模組：

| 模組 | 內容 |
|------|------|
| ⚙️ **固體密度實驗** | 鋁、鐵、金三種金屬，各 5 個待測物 |
| 💧 **液體密度實驗** | 水、食用油兩種液體，模擬去皮秤重 |

**核心概念：** 密度公式 D = M / V（質量 ÷ 體積）

---

## ⚙️ 固體密度實驗 — 操作說明

### 實驗目的
透過天平量質量、排水法量體積，計算金屬密度，並以散點圖驗證結果。

### 操作步驟

**① 選擇金屬種類**
- 頁面上方有三個按鈕：**🔘 鋁 / ⚙️ 鐵 / 🏅 金**
- 點選要測量的金屬種類

**② 拖曳金屬塊到天平**
- 從左側「待測物貨架」，用滑鼠**拖曳**金屬塊到中央的**電子天平**
- 天平螢幕會顯示質量讀數（公克）
- 📝 將讀數手動填入右側「實驗紀錄表」的 **M（g）** 欄位

**③ 拖曳金屬塊到量筒**
- 將同一個金屬塊繼續拖曳到右側的**量筒**
- 量筒水位會上升，顯示新的總體積
- 體積 V = 總體積 − 初始水位（100 mL）
- 📝 將體積讀數填入「實驗紀錄表」的 **V（cm³）** 欄位

**④ 自行計算密度**
- 用公式 **D = M ÷ V** 手動計算密度
- 填入完 M 和 V 後，按「**顯示 D**」按鈕驗證您的計算

**⑤ 重複實驗**
- 按「↺ 取回」將金屬塊歸位
- 對同種金屬的其他待測物重複步驟 ②～④
- 建議完成 5 個待測物，以獲得更準確的散點圖

**⑥ 查看散點圖**
- 右側會自動生成「質量對體積的關係圖」
- 點選上方的「**趨勢線**」按鈕，可加入各金屬的趨勢線
- 趨勢線的**斜率 = 該金屬密度 D**

### 參考密度值
| 金屬 | 標準密度 |
|------|---------|
| 鋁 | 2.70 g/cm³ |
| 鐵 | 7.87 g/cm³ |
| 金 | 19.30 g/cm³ |

## 💧 液體密度實驗 — 操作說明

### 實驗目的
模擬將量筒放在天平上去皮後，加入不同體積的液體，記錄質量與體積的關係。

### 操作步驟

**① 選擇液體種類**
- 頁面上方選擇：**💧 水** 或 **🫙 食用油**

**② 歸零（去皮）**
- 初始狀態：量筒放在天平上，體積 V = 0，天平顯示量筒本身質量（20 g）
- 按下「**⊘ 歸零（去皮）**」按鈕，天平歸零
- 此後天平只顯示**液體的淨質量**，不含量筒重量
- 再按一次可**復原**，回到顯示總質量模式

**③ 加入液體**
- 按「**＋**」按鈕增加液體體積（每次 +10 cm³）
- 按「**－**」按鈕減少液體體積（每次 -10 cm³）
- 量筒動畫會同步顯示液位變化

**④ 讀取並記錄數據**
- 讀取天平顯示的 **M（g）**（去皮後的液體質量）
- 讀取量筒上方標示的 **V（cm³）**
- 📝 手動填入右側「實驗紀錄表」

**⑤ 自行計算密度**
- 用公式 **D = M ÷ V** 手動計算
- 填入完後按「**顯示 D**」按鈕驗證

**⑥ 收集多筆數據並作圖**
- 重複步驟 ③～⑤，建議記錄 5 筆不同體積的數據
- 點選「**趨勢線**」按鈕，查看散點圖的斜率（= 液體密度 D）

### 參考密度值
| 液體 | 標準密度 |
|------|---------|
| 水（25°C） | 1.000 g/cm³ |
| 食用油 | 0.60 g/cm³ |

## 📊 散點圖功能說明

| 功能 | 說明 |
|------|------|
| **自動更新** | 每填入一筆數據，散點圖立即更新 |
| **趨勢線** | 按鈕開啟/關閉，斜率即為測量密度 D |
| **虛線** | 代表該物質的標準密度值（參考用） |
| **Tooltip** | 滑鼠移到數據點，顯示 V、M、D 數值 |
| **多物質** | 固體實驗可同時顯示多種金屬的數據點 |
