## 📖 實驗簡介

本網頁提供兩個完整的密度實驗模組：

| 模組 | 內容 |
|------|------|
| ⚙️ **固體密度實驗** | 鋁、鐵、金、木頭四種物質，各 5 個待測物（含浮體實驗）|
| 💧 **液體密度實驗** | 水、食用油、酒精、水銀四種液體，模擬去皮秤重 |
| 🔍 **未知物密度實驗** | 九個未知物 A~I，隨機分配 3~5 種密度，自行分類作答 |

**核心概念：** 密度公式 D = M / V（質量 ÷ 體積）

---

## ⚙️ 固體密度實驗 — 操作說明

**① 選擇物質種類**　點選頁面上方：🪵 木頭 / 🔘 鋁 / ⚙️ 鐵 / 🏅 金

**② 拖曳待測物到天平**　讀取質量 M，手動填入紀錄表

**③ 拖曳待測物到量筒**　觀察水位上升，V = 上升後水位 − 100 mL，填入紀錄表

**④ 自行計算密度**　計算 D = M ÷ V，再按「**顯示 D**」驗證

**⑤ 重複實驗**　建議完成 5 筆數據，點選「**作圖**」按鈕查看散點圖斜率

> 🌊 **木頭浮體特別說明**：木頭密度 0.50 g/cm³，會漂浮！
> 測體積時需將**鐵塊（重物）**和木塊一起放入量筒使其沉底。
> 木塊體積 V = 水位上升量 − 鐵塊體積（10 cm³）

### 物質標準密度參考

| 物質 | 標準密度 |
|------|---------|
| 🪵 木頭 | 0.50 g/cm³（浮體）|
| 🔘 鋁 | 2.70 g/cm³ |
| ⚙️ 鐵 | 7.87 g/cm³ |
| 🏅 金 | 19.30 g/cm³ |

## 💧 液體密度實驗 — 操作說明

### 操作步驟

**① 選擇液體種類**
- 點選：**💧 水**、**🫙 食用油**、**🍶 酒精** 或 **⚗️ 水銀**

**② 歸零（去皮）**
- 初始狀態：V = 0，天平顯示空量筒質量（20 g）
- 按「**⊘ 歸零（去皮）**」，天平歸零
- 之後天平只顯示**液體淨質量 M**
- 再按一次可**復原**回總質量模式

**③ 輸入液體體積**
- 在輸入框直接填入體積數值（cm³）
- 量筒動畫同步顯示液位

**④ 讀取並記錄數據**
- 📝 讀取天平的 **M（g）** 與量筒的 **V（cm³）**，填入紀錄表

**⑤ 自行計算密度並驗證**
- 計算 **D = M ÷ V**，再按「**顯示 D**」驗證

**⑥ 收集 5 筆數據，查看散點圖趨勢線**

### 液體標準密度參考

| 液體 | 標準密度 |
|------|---------|
| 💧 水 | 1.000 g/cm³ |
| 🫙 食用油 | 0.60 g/cm³ |
| 🍶 酒精 | 0.80 g/cm³ |
| ⚗️ 水銀 | 13.6 g/cm³ |

## 🔍 未知物密度實驗 — 操作說明

**① 選擇液體**　點選 💧 水 或 🫙 食用油（影響浮沉判斷）

**② 量測質量**　拖曳未知物 A~I 到**天平**（一次只能放 1 個），讀取 M

**③ 量測體積**　拖曳到**量筒**（最多 2 個），觀察浮沉現象，讀取水位變化計算 V

> 若物體漂浮，需加入**重物**一起放入量筒使其沉底，再扣除重物體積（10 cm³）

**④ 填入紀錄表**　手動輸入 M 和 V，按「**顯示 D**」驗算

**⑤ 觀察散點圖**　按「**📈 顯示作圖線**」，同一條線上的點代表同種物質

**⑥ 作答並對答案**
- 勾選哪些物體在液體中**漂浮**、哪些**沉底**
- 選擇九個物體共有幾種不同物質（3～5 種）
- 按「**🔍 對答案**」查看正確結果
