## 📖 實驗簡介

本網頁提供兩個完整的密度實驗模組：

| 模組 | 內容 |
|------|------|
| ⚙️ **固體密度實驗** | 鋁、鐵、金三種金屬，各 5 個待測物 |
| 💧 **液體密度實驗** | 水、食用油兩種液體，模擬去皮秤重 |

**核心概念：** 密度公式 D = M / V（質量 ÷ 體積）

---

## ⚙️ 固體密度實驗 — 操作說明

### 操作步驟

**① 選擇金屬種類**
- 頁面上方點選：**🔘 鋁 / ⚙️ 鐵 / 🏅 金**

**② 拖曳金屬塊到天平**
- 從「待測物貨架」拖曳金屬塊到**電子天平**
- 天平顯示質量讀數
- 📝 手動填入紀錄表的 **M（g）** 欄位

**③ 拖曳金屬塊到量筒**
- 將同一金屬塊拖曳到**量筒**，水位上升
- 體積 V = 上升後水位 − 初始水位（100 mL）
- 📝 手動填入紀錄表的 **V（cm³）** 欄位

**④ 自行計算密度**
- 用公式 **D = M ÷ V** 手動計算
- 填入完 M 和 V 後，按「**顯示 D**」按鈕驗證

**⑤ 重複實驗，建議完成 5 筆數據**
- 按「↺ 取回」將金屬塊歸位，換下一個繼續量測

**⑥ 查看散點圖**
- 點選「**趨勢線**」按鈕，趨勢線斜率即為該金屬密度 D

### 金屬標準密度參考

| 金屬 | 標準密度 |
|------|---------|
| 🔘 鋁 | 2.70 g/cm³ |
| ⚙️ 鐵 | 7.87 g/cm³ |
| 🏅 金 | 19.30 g/cm³ |

## 💧 液體密度實驗 — 操作說明

### 操作步驟

**① 選擇液體種類**
- 點選：**💧 水** 或 **🫙 食用油**

**② 歸零（去皮）**
- 初始狀態：V = 0，天平顯示空量筒質量（20 g）
- 按「**⊘ 歸零（去皮）**」，天平歸零
- 之後天平只顯示**液體淨質量 M**
- 再按一次可**復原**回總質量模式

**③ 輸入液體體積**
- 在輸入框直接填入體積數值（cm³）
- 量筒動畫同步顯示液位

**④ 讀取並記錄數據**
- 📝 讀取天平的 **M（g）** 與量筒的 **V（cm³）**，填入紀錄表

**⑤ 自行計算密度並驗證**
- 計算 **D = M ÷ V**，再按「**顯示 D**」驗證

**⑥ 收集 5 筆數據，查看散點圖趨勢線**

### 液體標準密度參考

| 液體 | 標準密度 |
|------|---------|
| 💧 水 | 1.000 g/cm³ |
| 🫙 食用油 | 0.60 g/cm³ |

| 功能 | 說明 |

| **趨勢線按鈕** | 開啟後顯示趨勢線，斜率 = 密度 D |
